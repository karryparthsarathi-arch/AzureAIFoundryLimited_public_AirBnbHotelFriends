# AirBnb OurFriends Hotel AI Assistant – Azure AI Foundry Agent

## Project Overview

AirBnb OurFriends Hotel AI Assistant is an Azure AI Foundry-powered guest support agent designed to answer hotel-related questions using only approved hotel knowledge documents such as Hotel Policies, Restaurant Menus, Spa Service List, Local Area Guide, and FAQ Compilation. The assistant follows a retrieval-first approach, cites source documents in every response, prioritizes hotel policy documents for rule-based questions, and avoids hallucinated answers by limiting responses to the uploaded knowledge base. This project demonstrates enterprise-grade AI assistant design using Azure AI Foundry Agents, grounded retrieval, responsible AI guardrails, source citation, and guest-friendly response formatting.

## Key Features

- Azure AI Foundry Agent integration
- Retrieval-grounded hotel guest assistant
- Source citation enforcement
- Hotel policy prioritization
- Friendly guest-service response style
- Fallback response when information is not found
- Config-driven Python implementation
- Portfolio-ready structure for GitHub and LinkedIn

## Use Case

This agent can be used by hotels, resorts, serviced apartments, or hospitality teams to provide guests with quick, consistent, and document-grounded answers about check-in rules, cancellation policy, restaurant menus, spa services, nearby attractions, and frequently asked questions.
